## Data to use

The following examples use publicly available datasets, and no data download is required to run them.

Data are accessed from two repositories:


| Service | Access Point | 
| ------ | ----------- | 
| <img src="https://storage.googleapis.com/hydroshare-prod-static-media/static/img/logo-lg.cf4395806c8e.png" width="100"> | [![Hydroshare Dataset](https://img.shields.io/badge/Hydroshare-Dataset-blue)](https://www.hydroshare.org/resource/6be754bf29cb488b815810c35f3f0ac9/) | 
| <img src="https://gdex.ucar.edu/media/images/GDEX_Logo.width-400.png" width="100"> | [![GDEX Dataset](https://img.shields.io/badge/GDEX-Dataset-blue)](https://gdex.ucar.edu/datasets/) | 


MPAS-A datasets are retrieved through the **THREDDS** data service.

<img src="https://www.unidata.ucar.edu/sites/default/files/styles/medium/public/images/thredds_tds-150x150.png" width="100">


The examples include two hurricane case studies:

1. A dataset developed at **Stevens Institute of Technology** for **Hurricane Ida (September 2021)** during its passage through New York.
2. A dataset prepared for the **MAAG Group** for **Hurricane Maria (September 2017)**.
